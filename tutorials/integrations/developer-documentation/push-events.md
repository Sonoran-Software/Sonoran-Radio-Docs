# Push Events

## Play Tone

The `play_tone` push event is sent to the game server when a dispatcher is sending a tone to one or more in-game speakers.

```json
{
    "id": "", // Community ID
    "key": "", // Community API Key
    "data": [
        {
            
        }
    ]
}
```
